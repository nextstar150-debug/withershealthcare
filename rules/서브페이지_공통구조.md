# 서브페이지 공통 구조 가이드
> 모든 서브페이지 작업 시 @서브페이지_공통구조.md 로 참조할 것.
> 브랜드소개 페이지는 히어로 영역 예외 적용 (별도 디자인).

---

## 📐 1. 서브 히어로 영역 (전 서브페이지 공통)

브랜드소개를 **제외한** 모든 서브페이지 최상단에 동일한 구조 적용.

### 구조
```html
<!-- 서브 히어로 영역 -->
<section class="sub-hero">
  <!-- 배경 비주얼 -->
  <div class="sub-hero__bg">
    <img
      src="./assets/sub-hero-bg.webp"
      alt=""
      width="1920"
      height="300"
      fetchpriority="high"
    >
  </div>

  <!-- 페이지 제목 -->
  <div class="sub-hero__content">
    <h1 class="sub-hero__title">회사개요</h1>

    <!-- 브레드크럼 -->
    <nav class="breadcrumb" aria-label="현재 위치">
      <ol class="breadcrumb__list">
        <li class="breadcrumb__item">
          <a href="./index.html" class="breadcrumb__link" aria-label="홈">
            <img src="./assets/icon-home.svg" alt="홈" width="16" height="16">
          </a>
        </li>
        <li class="breadcrumb__item">
          <span class="breadcrumb__text">회사소개</span>
        </li>
        <li class="breadcrumb__item breadcrumb__item--current">
          <span class="breadcrumb__text">회사개요</span>
        </li>
      </ol>
    </nav>
  </div>
</section>
```

### CSS
```css
/* 서브 히어로 */
.sub-hero {
  position: relative;
  display: flex;
  align-items: center;
  justify-content: center;
  text-align: center;
}

.sub-hero__bg {
  position: absolute;
  inset: 0;
  z-index: 0;
}

.sub-hero__bg img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

.sub-hero__content {
  position: relative;
  z-index: 1;
}

/* 브레드크럼 */
.breadcrumb__list {
  display: flex;
  align-items: center;
  gap: 8px;
  list-style: none;
  padding: 0;
  margin: 0;
  justify-content: center;
}

.breadcrumb__item + .breadcrumb__item::before {
  content: '·';
  margin-right: 8px;
}
```

### 규칙
- 배경 이미지는 전 서브페이지 **동일한 이미지** 사용 (`sub-hero-bg.webp`)
- `<h1>` 텍스트만 각 페이지에 맞게 변경
- 브레드크럼 텍스트도 각 페이지에 맞게 변경
- 브랜드소개 페이지는 이 구조 적용 **금지** (별도 디자인 시안 따를 것)

## ⚠️ 2. 서브 히어로 하단 간격 예외 규칙

> 전체 섹션 간격(160px / 80px)과 **별도**로 적용되는 예외입니다.

서브 히어로 영역 바로 아래 **첫 번째 콘텐츠 섹션**의 상단 간격만 아래 기준으로 적용.
나머지 섹션 간 간격은 기존 160px / 80px 규칙 그대로 유지.

| 브레이크포인트 | 간격 |
|---|---|
| 1024px 초과 | `120px` |
| 1024px 이하 | `60px` |

```css
/* 서브 히어로 → 첫 번째 콘텐츠 간격 예외 */
.sub-hero + .section {
  margin-top: 120px;
  padding-top: 0; /* Kombai 생성 padding-top 초기화 — margin과 중복 방지 */
}

@media (max-width: 1024px) {
  .sub-hero + .section {
    margin-top: 60px;
    padding-top: 0;
  }
}
```

> ✅ `.sub-hero + .section` 으로 첫 번째 섹션만 정확히 타겟팅되므로
> 나머지 `.section + .section` 규칙에는 영향 없음.
> ⚠️ Kombai가 섹션에 padding-top을 자동 생성하는 경우가 있으므로 반드시 0으로 초기화할 것.

## 📑 3. 탭 메뉴 (회사소개 4개 페이지 공통)

회사소개 하위 4개 페이지(회사개요 / 핵심가치 / 연혁 / 오시는길)에 동일하게 적용.
탭 클릭 시 해당 페이지로 이동. 현재 페이지 탭 자동 활성화.

### 구조
```html
<!-- 회사소개 탭 메뉴 -->
<nav class="tab-menu" aria-label="회사소개 메뉴">
  <ul class="tab-menu__list">
    <li class="tab-menu__item">
      <a href="./about.html" class="tab-menu__link">회사개요</a>
    </li>
    <li class="tab-menu__item">
      <a href="./core-values.html" class="tab-menu__link">핵심가치</a>
    </li>
    <li class="tab-menu__item">
      <a href="./history.html" class="tab-menu__link">연혁</a>
    </li>
    <li class="tab-menu__item">
      <a href="./directions.html" class="tab-menu__link">오시는길</a>
    </li>
  </ul>
</nav>
```

### CSS
```css
/* 탭 메뉴 */
.tab-menu__list {
  display: flex;
  list-style: none;
  padding: 0;
  margin: 0;
}

.tab-menu__link {
  display: block;
  padding: 16px 40px;
  text-decoration: none;
  transition: background-color var(--transition-base),
              color var(--transition-base);
}

/* 활성 탭 — 색상은 프로젝트 가이드 수령 후 적용 */
.tab-menu__link.is-active {
  /* color: var(--color-primary); */
  /* background-color: var(--color-primary); */
}
```

### JS — 현재 페이지 탭 자동 활성화
```javascript
// 탭 메뉴 자동 활성화
document.addEventListener('DOMContentLoaded', () => {
  const currentPath = window.location.pathname;

  document.querySelectorAll('.tab-menu__link').forEach(link => {
    if (!link) return;
    const href = link.getAttribute('href');
    if (!href) return;

    if (currentPath.includes(href.replace('./', '').replace('.html', ''))) {
      link.classList.add('is-active');
    }
  });
});
```

### 규칙
- 탭 메뉴는 회사소개 4개 페이지 **모두 동일**하게 삽입
- 클릭 시 페이지 이동 (SPA 방식 금지, 순수 href 이동)
- 현재 페이지 탭에 `is-active` 클래스 자동 부여
- 파일명은 `페이지_파일목록.md` 기준으로 href 매핑

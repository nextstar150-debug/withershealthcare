## ✏️ 작성 방법

1. Kombai 출력 완료 후 생성된 파일명 확인
2. 위 표 파일명 칸에 실제 파일명 입력
3. Cursor에 아래 명령어로 전달

```
@_rules/페이지_파일목록.md @_rules/공통요청_프롬프트.md
위 파일목록 기준으로 GNB href 연결하고 is-active 적용해줘
```

---

## ⚠️ 주의사항

- 파일명은 Kombai가 생성한 것을 그대로 사용 (임의 변경 금지)
- 파일명 변경이 필요한 경우 HTML 내 href도 함께 수정할 것
- 새 페이지 추가 시 표에 즉시 업데이트할 것
